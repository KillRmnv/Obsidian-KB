 # 📖 Полный гайд по Neovim + ваша конфигурация

## 🎮 Режимы работы

| Режим | Вход | Выход | Назначение |
|-------|------|-------|------------|
| **Normal** | `Esc`, `Ctrl+[` или `Ctrl+c` | `i`, `a`, `o`, `I`, `A`, `O` | Навигация и команды |
| **Insert** | `i`, `a`, `o`, `I`, `A`, `O`, `s`, `S`, `c`, `C` | `Esc` или `Ctrl+[` | Ввод текста |
| **Visual** | `v` (символы), `V` (строки), `Ctrl+v` (блок) | `Esc` | Выделение текста |
| **Visual Line** | `V` | `Esc` | Выделение строк |
| **Visual Block** | `Ctrl+v` | `Esc` | Блочное выделение |
| **Command** | `:` | `Enter` или `Esc` | Ввод команд |
| **Replace** | `R` | `Esc` | Замена текста |

## 🧭 Навигация (Normal режим)

### Базовые перемещения
| Клавиша | Действие | Пример |
|---------|----------|--------|
| `h` | Влево | |
| `j` | Вниз | `5j` — 5 строк вниз |
| `k` | Вверх | `10k` — 10 строк вверх |
| `l` | Вправо | |
| `w` | Слово вперёд (начало) | `3w` — 3 слова |
| `W` | Слово вперёд (большое, с пунктуацией) | |
| `b` | Слово назад (начало) | |
| `B` | Слово назад (большое) | |
| `e` | Конец слова | |
| `E` | Конец большого слова | |
| `ge` | Конец предыдущего слова | |

### Перемещение по строке
| Клавиша | Действие |
|---------|----------|
| `0` | Начало строки |
| `^` | Первый непробельный символ |
| `$` | Конец строки |
| `g_` | Последний непробельный символ |
| `|` | Столбец N (по умолчанию 1) |
| `f{char}` | На символ вперёд (Flit: `s{char}{char}`) |
| `F{char}` | На символ назад |
| `t{char}` | Перед символом вперёд |
| `T{char}` | После символа назад |
| `;` | Повторить последний f/F/t/T |
| `,` | Повторить в обратную сторону |

### Перемещение по файлу
| Клавиша | Действие |
|---------|----------|
| `gg` | Начало файла |
| `G` | Конец файла |
| `:{n}` | Перейти на строку N (например, `:25`) |
| `{` | Предыдущий параграф |
| `}` | Следующий параграф |
| `(` | Предыдущее предложение |
| `)` | Следующее предложение |
| `[[` | Предыдущая секция/функция |
| `]]` | Следующая секция/функция |
| `[]` | Предыдущая закрывающая скобка |
| `][` | Следующая закрывающая скобка |
| `%` | К парной скобке |

### Прокрутка

| -------- | --------------------------- |
| `Ctrl+e` | Прокрутка вниз на строку    |
| `Ctrl+y` | Прокрутка вверх на строку   |
| `Ctrl+d` | Прокрутка вниз (полэкрана)  |
| `Ctrl+u` | Прокрутка вверх (полэкрана) |
| `zz`     | Центрировать курсор         |
| `zt`     | Курсор вверх экрана         |
| `zb`     | Курсор вниз экрана          |
| `H`      | Перейти на верх экрана      |
| `M`      | Перейти на середину экрана  |
| `L`      | Перейти на низ экрана       |

### Прыжки (Marks)
| Клавиша | Действие |
|---------|----------|
| `m{a-zA-Z}` | Установить метку |
| `` `a `` | Перейти к метке a (точная позиция) |
| `'a` | Перейти к строке метки a |
| `` `` `` | Вернуться к месту перед прыжком |
| `''` | Вернуться к строке перед прыжком |
| `` `. `` | К месту последнего изменения |
| `'^` | К месту последнего вставки |

## ✏️ Редактирование

### Вход в Insert режим
| Клавиша | Действие |
|---------|----------|
| `i` | Вставка перед курсором |
| `I` | Вставка в начало строки |
| `a` | Вставка после курсора |
| `A` | Вставка в конец строки |
| `o` | Новая строка ниже |
| `O` | Новая строка выше |
| `s` | Удалить символ и вставить |
| `S` | Удалить строку и вставить |
| `c{motion}` | Изменить (delete + insert) |
| `C` | Изменить до конца строки |
| `cc` или `S` | Изменить всю строку |

### Удаление (Delete)
| Клавиша | Действие | Пример |
|---------|----------|--------|
| `x` | Удалить символ под курсором | `5x` — 5 символов |
| `X` | Удалить символ перед курсором | |
| `d{motion}` | Удалить по motion | `dw` — слово, `d$` — до конца |
| `dd` | Удалить строку | `3dd` — 3 строки |
| `D` | Удалить до конца строки (`d$`) |
| `di"` | Удалить внутри кавычек |
| `da"` | Удалить кавычки и содержимое |
| `diw` | Удалить внутреннее слово |
| `daw` | Удалить слово с пробелом |
| `di(` или `dib` | Удалить внутри скобок |
| `di[` | Удалить внутри квадратных скобок |
| `di{` или `diB` | Удалить внутри фигурных скобок |
| `dit` | Удалить внутри HTML-тега |
| `diw` | Удалить внутри слова |

### Копирование (Yank)
| Клавиша | Действие | Пример |
|---------|----------|--------|
| `y{motion}` | Копировать | `yw` — слово |
| `yy` или `Y` | Копировать строку | `3yy` — 3 строки |
| `yi"` | Копировать внутри кавычек |
| `ya"` | Копировать с кавычками |
| `yiw` | Копировать внутреннее слово |
| `yaw` | Копировать слово с пробелом |
| `y$` | Копировать до конца строки |
| `y^` или `y0` | Копировать до начала |

### Вставка (Put)
| Клавиша | Действие |
|---------|----------|
| `p` | Вставить после курсора/ниже |
| `P` | Вставить перед курсором/выше |
| `gp` | Вставить и курсор после |
| `gP` | Вставить перед и курсор после |
| `]p` | Вставить с отступом |
| `[p` | Вставить перед с отступом |

### Изменение регистра
| Клавиша | Действие |
|---------|----------|
| `~` | Изменить регистр символа |
| `g~{motion}` | Изменить регистр по motion |
| `g~~` или `g~g~` | Изменить регистр строки |
| `gu{motion}` | В нижний регистр |
| `gU{motion}` | В ВЕРХНИЙ регистр |
| `gugu` или `guu` | Строку в нижний |
| `gUgU` или `gUU` | Строку в ВЕРХНИЙ |

### Отмена и повтор
| Клавиша | Действие |
|---------|----------|
| `u` | Отменить (Undo) |
| `U` | Отменить изменения в строке |
| `Ctrl+r` | Повторить (Redo) |
| `.` | Повторить последнее действие |
| `@:` | Повторить последнюю команду |
| `&` | Повторить последнюю замену |

## 🔍 Поиск и замена

### Поиск
| Клавиша | Действие |
|---------|----------|
| `/pattern` | Искать вперёд |
| `?pattern` | Искать назад |
| `n` | Следующее совпадение |
| `N` | Предыдущее совпадение |
| `*` | Искать слово под курсором вперёд |
| `#` | Искать слово под курсором назад |
| `g*` | Частичное совпадение вперёд |
| `g#` | Частичное совпадение назад |
| `:noh` | Убрать подсветку поиска |

### Замена
| Команда | Действие |
|---------|----------|
| `:%s/old/new/` | Заменить первое вхождение в каждой строке |
| `:%s/old/new/g` | Заменить все в файле |
| `:%s/old/new/gc` | Заменить с подтверждением |
| `:%s/old/new/gi` | Заменить без учёта регистра |
| `:s/old/new/g` | Заменить все в текущей строке |
| `:'<,'>s/old/new/g` | Заменить в выделенном |
| `:%s//new/g` | Использовать последний поиск |

|---------|----------|

| Клавиша | Действие |
|---------|----------|

| `Ctrl+w _` | Максимальная высота |

## 🆘 Экстренная помощь

| Проблема | Решение |
|----------|---------|
| Застрял в Insert режиме | `Esc` или `Ctrl+[` или `Ctrl+c` |
| Не могу выйти | `Esc` → `:qa!` |
| Случайно удалил текст | `u` для отмены |
| Всё сломалось | `Esc` несколько раз → `:q!` |
| Зависло | `Ctrl+c` или `Ctrl+[` |
| Не работает поиск (подсветка) | `:noh` (no highlight) |
| Не вижу диагностики | `:Trouble diagnostics toggle` или `<leader>xx` |

## 📋 Шпаргалка по комбинациям

### Частые комбинации
```
ci"  - Изменить текст внутри кавычек
yi(  - Скопировать внутри скобок
diw  - Удалить слово
yiw  - Скопировать слово
ddp  - Поменять строки местами (удалить и вставить ниже)
xp   - Поменять символы местами
>>   - Увеличить отступ
<<   - Уменьшить отступ
==   - Автоотступ
gf   - Перейти к файлу под курсором
gx   - Открыть URL под курсором
```

### Макросы
```
qa   - Начать запись макроса в регистр a
q    - Остановить запись
@a   - Воспроизвести макрос a
@@   - Повторить последний макрос
5@a  - Воспроизвести макрос 5 раз
```

### Регистры
```
"ayy  - Скопировать строку в регистр a
"ap   - Вставить из регистра a
"+yy  - Скопировать в системный буфер
"+p   - Вставить из системного буфера
"0p   - Вставить последнее копирование (не затронуто удалением)
```
