Вот **базовые горячие клавиши** для работы с окнами в **Hyprland** (с учётом установленных вами **KooL's Dotfiles**).

> ⚠️ **`SUPER` = клавиша `Win`** (⊞) на клавиатуре.

---

## 🪟 Основные действия с окнами

| Комбинация | Действие |
|------------|----------|
| `SUPER + Q` | ❌ Закрыть активное окно |
| `SUPER + C` | 🔄 Перезагрузить конфиг (после правок!) |
| `SUPER + M` | 🚪 Выйти из сессии Hyprland |
| `SUPER + E` | 📁 Открыть файловый менеджер (Thunar) |
| `SUPER + T` | 💻 Открыть терминал (Kitty) |
| `SUPER + Shift + E` | 🌐 Открыть браузер |

---

## 🎯 Переключение фокуса между окнами

| Комбинация | Действие |
|------------|----------|
| `SUPER + ←` / `→` / `↑` / `↓` | Переместить фокус на окно в направлении стрелки |
| `SUPER + h` / `l` / `k` / `j` | То же самое (в стиле Vim) |
| `SUPER + Tab` | 🔄 Переключиться на следующее окно (циклически) |
| `SUPER + Shift + Tab` | 🔄 Переключиться на предыдущее окно |

---

## 🧱 Перемещение окон

| Комбинация | Действие |
|------------|----------|
| `SUPER + Shift + ←` / `→` / `↑` / `↓` | Переместить активное окно в направлении стрелки |
| `SUPER + Shift + h` / `l` / `k` / `j` | То же самое (в стиле Vim) |
| `SUPER + Shift + F` | 🪟 Развернуть окно на весь экран (fullscreen) |
| `SUPER + Shift + M` | 📐 Развернуть окно в "псевдо-полный" режим (maximize) |
| `SUPER + Shift + B` | 🪟 Переключить окно в плавающий режим (floating) / обратно |

---

## 📐 Изменение размера окон (в тайлинге)

| Комбинация | Действие |
|------------|----------|
| `SUPER + R` | ✋ Войти в режим изменения размера (курсор изменится) |
| `→` / `←` / `↑` / `↓` | Изменить размер активного окна в режиме `SUPER+R` |
| `ESC` или `SUPER + R` (повторно) | Выйти из режима изменения размера |

> 💡 **Совет:** В тайлинговом менеджере окна обычно "подгоняются" автоматически. Ручное изменение размера нужно редко.

---

## 🗂️ Работа с рабочими столами (воркспейсами)

| Комбинация | Действие |
|------------|----------|
| `SUPER + 1` / `2` / `3` ... `0` | Перейти на воркспейс №1, 2, 3... |
| `SUPER + Shift + 1` / `2` / `3` ... | Переместить активное окно на воркспейс №1, 2, 3... |
| `SUPER + Ctrl + ←` / `→` | Перейти на предыдущий / следующий воркспейс |
| `SUPER + Mouse Wheel` | 🔄 Прокрутка воркспейсов колесом мыши |

---

## 🖱️ Управление мышью

| Действие | Комбинация |
|----------|-----------|
| **Перемещение окна** | `SUPER + ЛКМ` (зажать и тянуть окно) |
| **Изменение размера окна** | `SUPER + ПКМ` (зажать и тянуть край окна) |
| **Фокус на окно** | Просто клик ЛКМ (без модификаторов) |

> 💡 Это очень удобно: не нужно запоминать клавиши для каждого действия — просто держите `Win` и управляйте мышью.

---

## 🔍 Специальные функции (из вашего пакета дотфилов)

| Комбинация | Действие |
|------------|----------|
| `SUPER + Shift + S` | 📸 Сделать скриншот области (через `grim` + `slurp`) |
| `SUPER + Shift + W` | 🎨 Выбрать обои (через рофи-меню) |
| `SUPER + Shift + T` | 🎨 Переключить тему (цвета, иконки) |
| `SUPER + Space` | 🚀 Открыть меню приложений (Rofi / Wofi) |
| `SUPER + /` | ❓ Открыть справку по горячим клавишам (KeyHints) |
| `SUPER + Shift + O` | 🗂️ Открыть обзор воркспейсов (Overview, если включён) |
| `SUPER + Shift + L` | 🔒 Заблокировать экран (hyprlock) |

## 🔧 Как посмотреть ВСЕ горячие клавиши в вашей системе

### Способ 1: Через встроенную справку (если есть)
Нажмите:
```
SUPER + /
```
Или запустите скрипт вручную:
```bash
~/.config/hypr/scripts/KeyHints.sh
```

### Способ 2: Посмотреть файл конфигурации
```bash
# Основные пользовательские клавиши:
cat ~/.config/hypr/UserConfigs/UserKeybinds.conf

# Системные клавиши (не редактируйте, только для ознакомления):
cat ~/.config/hypr/configs/Keybinds.conf
```

### Способ 3: Через терминал (hyprctl)
```bash
# Вывести все активные бинды в формате JSON:
hyprctl binds -j | jq

# Или просто текстом:
hyprctl binds
```

> 💡 Установите `jq` для красивого вывода: `sudo apt install jq`

## ✏️ Как добавить СВОЮ горячую клавишу

1.  Откройте файл пользовательских клавиш:
    ```bash
    nano ~/.config/hypr/UserConfigs/UserKeybinds.conf
    ```

2.  Добавьте строку в формате:
    ```ini
    # Синтаксис: bind = МОДИФИКАТОР, КЛАВИША, ДЕЙСТВИЕ, ПАРАМЕТР

    # Запустить калькулятор:
    bind = SUPER, K, exec, gnome-calculator

    # Переключить раскладку клавиатуры:
    bind = SUPER, Space, exec, hyprctl switchxkblayout

    # Убить процесс (если окно не отвечает):
    bind = SUPER + SHIFT, K, exec, pkill -f "$(hyprctl activewindow -j | jq -r '.title')"
    ```

3.  Сохраните: `Ctrl+O` → `Enter` → `Ctrl+X`

4.  Примените изменения:
    ```
    SUPER + Shift + C
    ```

## 🆘 Если клавиши не работают

1.  **Проверьте, не переопределена ли клавиша** в другом файле:
    ```bash
    grep -r "bind = SUPER, Q" ~/.config/hypr/
    ```

2.  **Посмотрите логи** на предмет ошибок:
    ```bash
    journalctl -b | grep -i "hypr" | tail -30
    ```

3.  **Проверьте, не блокирует ли другая программа** клавишу (например, GNOME Shell или расширение).

4.  **Перезагрузите сессию** полностью (логин/логаут), если `hyprctl reload` не помог.
