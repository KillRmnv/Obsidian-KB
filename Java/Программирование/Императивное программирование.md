Подход, при котором программа описывает **как именно** должна быть выполнена задача.

Код задаёт последовательность команд, изменяющих состояние программы: используются присваивания, циклы, условные операторы и явное управление потоком выполнения. Разработчик напрямую управляет шагами алгоритма и состоянием данных.

```python
numbers = [1, 2, 3, 4, 5, 6]
result = []

for n in numbers:
    if n % 2 == 0:
        result.append(n * n)

print(result)
```
