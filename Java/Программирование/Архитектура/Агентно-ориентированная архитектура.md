Подход к построению систем, при котором система рассматривается как **совокупность автономных агентов**, взаимодействующих между собой и с окружающей средой для достижения целей.

**Особенности агентов:**

- **Автономность** - агент может принимать решения самостоятельно.
- **Реактивность** - агент реагирует на изменения окружающей среды.
- **Проактивность** - агент способен ставить и достигать собственные цели.
- **Взаимодействие** - агенты общаются друг с другом через сообщения, координируют действия.

**Применение:**

- Сложные распределённые системы.
- Робототехника и интеллектуальные системы.
- Моделирование поведения сложных систем и многопользовательских сред.
