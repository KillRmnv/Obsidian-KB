# ☕ J2SE 1.3 — _“Kestrel”_ (Java 2 Platform, Standard Edition 1.3)

📅 **Дата выхода:** 8 мая 2000 года  
📦 **Кодовое имя:** _Kestrel_  
🔧 **Основной фокус:** производительность, стабильность и модернизация JVM

---

## 🌍 Общая характеристика

**Java 2 Standard Edition 1.3 (J2SE 1.3)** — это не столько революционное обновление, сколько **“полировка” и ускорение** Java-платформы после масштабных изменений в 1.2.

Главная цель: сделать Java промышленно надёжной, быстрой и интегрированной с реальными системами.  
Релиз также закрепил переход на **HotSpot JVM**, впервые ставший стандартным движком исполнения.

---

## ⚙️ Ключевые нововведения

### 1️⃣ HotSpot JVM — новая виртуальная машина по умолчанию

- Главная инновация 1.3.
    
- **HotSpot** заменил старую классическую JVM, обеспечив:
    
    - Динамическую оптимизацию байткода.
        
    - Более умный JIT-компилятор.
        
    - Улучшенный сборщик мусора.
        
    - Адаптивную компиляцию (оптимизация "горячих" методов).
        

💡 Итог: производительность выросла **на 20–40%** в типичных задачах по сравнению с 1.2.

---

### 2️⃣ Java Sound API (`javax.sound`)

- Добавлен **новый звуковой API**, включающий:
    
    - Воспроизведение и запись звука (WAV, MIDI и др.).
        
    - Микширование и эффекты.
        
    - Поддержку музыкальных синтезаторов (MIDI Sequencer и Synthesizer).
        

📘 Пример:

```java
import javax.sound.sampled.*;
import java.io.File;

Clip clip = AudioSystem.getClip();
clip.open(AudioSystem.getAudioInputStream(new File("sound.wav")));
clip.start();
```

---

### 3️⃣ Java Naming and Directory Interface (JNDI) → часть стандартной платформы

- В 1.2 был добавлен как дополнительный пакет, а в 1.3 стал **встроенным** в J2SE.
    
- Позволяет работать с каталогами и службами имен (LDAP, DNS, RMI registry и т. д.).
    

---

### 4️⃣ CORBA и RMI-IIOP (улучшенная межплатформенность)

- Улучшена интеграция с CORBA через **RMI over IIOP**.
    
- Java 1.3 включала **Object Request Broker (ORB)** по умолчанию.
    
- Поддержка новых стандартов OMG IDL.
    

---

### 5️⃣ Java Platform Debugger Architecture (JPDA)

Новый стандарт отладки в Java:

- Три уровня:
    
    1. **JVMTI (Virtual Machine Tool Interface)** — низкоуровневое управление JVM.
        
    2. **JDWP (Debug Wire Protocol)** — протокол взаимодействия с отладчиком.
        
    3. **JDI (Java Debug Interface)** — API для IDE (используется в NetBeans, Eclipse).
        

💡 Именно JPDA позволила IDE впервые запускать и управлять отладкой Java-программ на уровне JVM.

---

### 6️⃣ Timer API

- Добавлен класс `java.util.Timer` и `TimerTask` для планирования фоновых задач.
    

📘 Пример:

```java
import java.util.*;

Timer timer = new Timer();
timer.schedule(new TimerTask() {
    public void run() {
        System.out.println("Tick");
    }
}, 0, 1000);
```

---

### 7️⃣ Java Naming Enhancements + RMI Improvements

- **RMI** теперь полностью основано на **JRMP (Java Remote Method Protocol)** и оптимизировано.
    
- Улучшена работа с динамическими прокси-классами (Dynamic Proxies).
    
- Улучшена поддержка загрузки классов по сети.
    

---

### 8️⃣ Улучшения AWT и Swing

- Swing стал стабильнее и быстрее.
    
- Улучшена поддержка двойной буферизации (меньше мерцаний при отрисовке).
    
- Добавлен **Java 2D printing API** (печать графики, изображений и текста).
    
- AWT получил улучшенные события клавиатуры и мыши, особенно для UNIX-платформ.
    

---

### 9️⃣ Networking API расширен

- Новый класс `java.net.InetAddress` теперь поддерживает **IPv6** (экспериментально).
    
- Добавлены улучшения для многопоточности и кэширования DNS-запросов.
    

---

### 🔟 Collections и Core API улучшения

- `Collections` и `ListIterator` получили исправления и оптимизации.
    
- Новый класс `StrictMath` — полностью детерминированные математические вычисления (для одинаковых результатов на разных JVM).
    
- Улучшен `System.getProperty()` и `Runtime.exec()`.

## 💡 Почему J2SE 1.3 важен

- Стабилизировал архитектуру, заложенную в 1.2.
    
- Сделал Java действительно **производительной и зрелой**.
    
- Впервые ввёл полноценную отладочную архитектуру (JPDA).
    
- Принёс мультимедиа (Java Sound) и улучшенную работу с сетью.
    
- Подготовил почву для появления **J2SE 1.4**, где уже появились `assert`, `NIO`, и `logging`.
