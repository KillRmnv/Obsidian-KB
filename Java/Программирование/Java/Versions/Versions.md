![[Pasted image 20251101133436.png]]### 🧭 Основные версии Java SE по годам:

| Версия               | Год выхода | Ключевые изменения                                                      |
| -------------------- | ---------- | ----------------------------------------------------------------------- |
| **JDK 1.0**          | 1996       | Первая версия Java                                                      |
| [[**JDK 1.1**]]      | 1997       | Введение inner-классов, JDBC                                            |
| [[**J2SE 1.2**]]     | 1998       | Swing, Collections Framework                                            |
| [[**J2SE 1.3**]]     | 2000       | RMI улучшения, HotSpot JVM                                              |
| **J2SE 1.4**         | 2002       | assert, NIO, logging API                                                |
| **Java SE 5 (1.5)**  | 2004       | Generics, annotations, autoboxing, enhanced for                         |
| **Java SE 6**        | 2006       | Стабильность, улучшения производительности                              |
| **Java SE 7**        | 2011       | try-with-resources, diamond operator, NIO.2                             |
| **Java SE 8**        | 2014       | Lambda-выражения, Stream API, Date/Time API                             |
| **Java SE 9**        | 2017       | Модульная система (Project Jigsaw)                                      |
| **Java SE 10**       | 2018       | `var` для локальных переменных                                          |
| **Java SE 11 (LTS)** | 2018       | LTS, удалены JavaFX и JAXB из JDK                                       |
| **Java SE 12–15**    | 2019–2020  | Switch expressions, text blocks, records (preview)                      |
| **Java SE 17 (LTS)** | 2021       | Pattern Matching, Sealed Classes, LTS                                   |
| **Java SE 18–20**    | 2022–2023  | Инкубаторские и preview-функции                                         |
| **Java SE 21 (LTS)** | 2023       | Virtual Threads (Project Loom), String Templates, Sequenced Collections |
| **Java SE 22–23**    | 2024–2025  | Улучшения виртуальных потоков, Record Patterns                          |
