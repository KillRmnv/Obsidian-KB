### Архитектура логгирования (Фасад + Реализация)

В современной Java-разработке используется связка из двух компонентов:

1. **Фасад (Интерфейс):** **SLF4J** (Simple Logging Facade for Java).
    - Вы пишете код против него.
    - Позволяет менять реализацию логгера без изменения кода приложения.
2. **Реализация (Движок):** **Logback** (стандарт для Spring Boot) или **Log4j2**.
    - Именно он пишет в файл, форматирует и ротирует логи.
