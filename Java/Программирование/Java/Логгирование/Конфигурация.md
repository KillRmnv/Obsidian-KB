### Конфигурация для Logback

Файл `src/main/resources/logback-spring.xml` (или `application.properties`).

**Пример `application.properties`:**
```
# Уровень логирования для всего приложения
logging.level.root=INFO

# Уровень для вашего пакета (включить DEBUG для отладки)
logging.level.com.example.myapp=DEBUG

# Уровень для Hibernate (см. ниже)
logging.level.org.hibernate.SQL=DEBUG
logging.level.org.hibernate.type.descriptor.sql.BasicBinder=TRACE

# Путь к файлу логов
logging.file.name=logs/app.log
```
