## Решения и гарантии

### 1. **synchronized** (решает все проблемы)
```java
class SynchronizedSolution {
    private int count = 0;
    
    public synchronized void increment() {
        count++; // Атомарно + visibility гарантии
    }
    
    public synchronized int getCount() {
        return count; // Гарантированно актуальное значение
    }
}
```

### 2. **volatile** (только visibility, не атомарность)
```java
class VolatileExample {
    private volatile boolean flag = false;
    private volatile int value = 0;
    
    // volatile обеспечивает:
    // 1. Видимость изменений
    // 2. Запрет reordering относительно других volatile операций
    // НЕ обеспечивает атомарность сложных операций!
}
```

### 3. **Atomic классы** (атомарность + visibility)
```java
import java.util.concurrent.atomic.*;

class AtomicSolution {
    private AtomicInteger count = new AtomicInteger(0);
    private AtomicBoolean initialized = new AtomicBoolean(false);
    
    public void safeIncrement() {
        count.incrementAndGet(); // Атомарно + CAS операция
    }
    
    public void init() {
        // compareAndSet обеспечивает атомарность
        if (initialized.compareAndSet(false, true)) {
            // Инициализация
        }
    }
}
```

### 4. **final поля** (безопасная публикация)
```java
class SafePublication {
    private final int immutableValue;
    private volatile boolean ready;
    
    public SafePublication(int value) {
        this.immutableValue = value; // Гарантированно видно после конструктора
        this.ready = true;           // volatile обеспечивает visibility
    }
}
```
