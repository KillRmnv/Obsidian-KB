## Основные команды

- `gradle build` — полная сборка с тестами и упаковкой.
    
- `gradle clean` — очистка build/.
    
- `gradle test` — запуск тестов.
    
- `gradle run` — запуск приложения (с application-плагином).
    
- `gradle tasks` — список задач.
    

Опции: `--refresh-dependencies` (обновить deps), `-x test` (пропустить тесты), `--parallel` (параллельная сборка), `--daemon` (фоновый демон для скорости).[](https://ru.hexlet.io/blog/posts/sozdanie-i-nastroyka-build-gradle-dlya-java-proektov-podrobnoe-rukovodstvo)​
