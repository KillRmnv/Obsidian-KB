``` groovy
plugins { id 'java' id 'application' } 
group = 'com.example' 
version = '1.0' 
repositories { mavenCentral() } 
dependencies { implementation 'org.apache.commons:commons-lang3:3.14.0' testImplementation 'org.junit.jupiter:junit-jupiter:5.10.3' 
testRuntimeOnly 'org.junit.platform:junit-platform-launcher' }
 application { mainClass = 'com.example.App' } 
 tasks.named('test') { useJUnitPlatform() }
```

**plugins {}**  
Активирует два ключевых плагина:

- `java` — добавляет компиляцию Java (compileJava), тесты (test), упаковку JAR (jar), source sets (src/main/java, src/test/java).[](https://docs.gradle.org/current/userguide/java_plugin.html)​
    
- `application` — расширяет для запуска приложения (run), создания стартовых скриптов (startScripts), дистрибутива (installDist). Автоматически применяет Distribution plugin.[](https://docs.gradle.org/current/userguide/application_plugin.html)​
    

**group и version**  
Определяют Maven-координаты артефакта: `com.example:проект:1.0`. Используется в именах JAR (com.example-1.0.jar) и публикации.[](https://www.baeldung.com/gradle-building-a-java-app)​

**repositories {}**  
Источник зависимостей — официальный Maven Central. Gradle скачивает оттуда библиотеки в локальный кэш (~/.gradle/caches).[](https://ru.hexlet.io/blog/posts/sozdanie-i-nastroyka-build-gradle-dlya-java-proektov-podrobnoe-rukovodstvo)​

**dependencies {}**

- `implementation 'org.apache.commons:commons-lang3:3.14.0'` — основная зависимость для продакшена (доступна в runtime classpath).
    
- `testImplementation 'org.junit.jupiter:junit-jupiter:5.10.3'` — только для тестов (src/test/java).
    
- `testRuntimeOnly 'org.junit.platform:junit-platform-launcher'` — runtime для запуска JUnit 5 на JVM.[](https://docs.gradle.org/current/userguide/java_plugin.html)​
    

**application {}**  
Указывает точку входа: `mainClass = 'com.example.App'` (класс с public static void main). Включает задачу `run` для `./gradlew run`.[](https://docs.gradle.org/current/userguide/application_plugin.html)​

**tasks.named('test') {}**  
Настраивает задачу test для JUnit 5: `useJUnitPlatform()` интегрирует JUnit Platform engine вместо legacy Vintage.[](https://docs.gradle.org/current/userguide/java_plugin.html)​

## Генерируемые задачи

После применения плагинов доступны команды (проверьте `./gradlew tasks`):

| Задача            | Что делает                           | Зависимости                                                                     |
| ----------------- | ------------------------------------ | ------------------------------------------------------------------------------- |
| `./gradlew build` | Полная сборка (compile + test + jar) | check, assemble [](https://docs.gradle.org/current/userguide/java_plugin.html)​ |
| `./gradlew run`   | Запуск App.main()                    | classes [](https://docs.gradle.org/current/userguide/application_plugin.html)​  |
| `./gradlew test`  | Unit-тесты                           | compileTestJava                                                                 |
| `./gradlew clean` |                                      |                                                                                 |
