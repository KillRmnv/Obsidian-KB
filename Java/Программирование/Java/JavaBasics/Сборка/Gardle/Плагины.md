## Зависимости и плагины

Зависимости в блоке dependencies:

groovy

`plugins {     id 'java'    id 'application' } repositories {     mavenCentral() } dependencies {     implementation 'org.springframework:spring-core:6.1.0'    testImplementation 'org.junit.jupiter:junit-jupiter:5.10.0' }`

Плагины активируют фазы: java-library, spring-boot. Версионирование через Version Catalog (gradle/libs.versions.toml) для мульти-модулей.
