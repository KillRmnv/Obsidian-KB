- Появились в Java 9 (Project Jigsaw).
    
- Это **единицы разбиения приложения на части**, более крупные, чем пакеты.
    
- Модуль может содержать **несколько пакетов** и управляет:
    
    - какие пакеты он **экспортирует** другим модулям;
        
    - от каких модулей он **зависит**.
      ```bash
      my-project/
 └── src/
      └── my.module/
           ├── module-info.java
           └── com/
               └── example/
                   └── app/
                       └──Main.java

      ```

    ```
    module my.module {
    requires java.sql;          // зависимость от стандартного модуля
    exports com.example.app;    // наружу доступен только этот пакет
}
  
    ```
