**Наследование** — это один из четырех столпов объектно-ориентированного программирования (ООП), который позволяет одному классу (подклассу) перенимать свойства и методы другого класса (суперкласса).

В Java наследование реализуется с помощью ключевого слова `extends`.

### Основные правила наследования

1. **Одиночное наследование:** В Java класс может наследоваться только от **одного** родителя. Это сделано для того, чтобы избежать «алмаза смерти» (конфликтов имен методов).
    
2. **Иерархия:** Все классы в Java неявно наследуются от базового класса `Object`.
    
3. **Что наследуется:** Подкласс получает все `public` и `protected` поля и методы родителя. `private` поля тоже наследуются, но доступ к ним возможен только через публичные методы (геттеры/сеттеры).
    
4. **Is-a связь:** Наследование описывает отношение «является». Например, `Dog extends Animal` (Собака _является_ Животным).

### Ключевое слово `super`

Слово `super` — это специальная ссылка на объект родительского класса. Она используется в трех основных случаях:

#### 1. Вызов конструктора родителя

Это **самое важное** правило: конструктор подкласса всегда должен вызывать конструктор родителя (явно или неявно).

- Если вы не напишете `super()`, Java сама вставит вызов пустого конструктора родителя первой строкой.
    
- Вызов `super(...)` должен быть **обязательно первой строкой** в конструкторе.
