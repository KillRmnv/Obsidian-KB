## Что делает `flatMap()`?

**`flatMap()`** — это операция, которая:

1. **Преобразует** каждый элемент стрима (как обычный `map()`)
    
2. **"Разворачивает" (flattens)** получившиеся стримы в один общий стрим
    

Проще говоря: **1 элемент → N элементов**
