└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│              1. КОНСТРУКТОР (Constructor)                       │

│         new MyServlet()                                         │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│              2. init(ServletConfig config)                      │
│         ═══════════════════════════════════                     │
│         • Инициализация ресурсов                                │
│         • Подключение к БД                                      │
│         • Загрузка конфигурации                                 │
│         • Создание пулов потоков                                │
│                                                                 │
│         ⚠️ Выполняется ТОЛЬКО ОДИН РАЗ за жизнь сервлета!       │
└─────────────────────────────────────────────────────────────────┘
                              ↓
                    ┌─────────────────┐
                    │  СЕРВЛЕТ ГОТОВ  │
                    │    К РАБОТЕ     │
                    │  (ready state)  │
                    └─────────────────┘
                              ↓
        ┌───────────────────────────────────────────────────┐
        │           3. service(request, response)           │
        │      ═══════════════════════════════════════      │
        │      • Вызывается для КАЖДОГО запроса             │
        │      • Многопоточность: каждый запрос в           │

        │        сервлета для всех запросов!                │
        │                                                     │
        │      request 1 ──→ service() ──→ response 1       │
        │      request 2 ──→ service() ──→ response 2  ← одновременно!
        │      request 3 ──→ service() ──→ response 3       │
        └───────────────────────────────────────────────────┘
                              ↓
        ┌───────────────────────────────────────────────────┐
        │      4. destroy()                                 │
        │      ═══════════════════════════════════════      │
        │      • Вызывается при остановке приложения        │
        │      • Освобождение ресурсов                      │
        │      • Закрытие соединений с БД                   │
        │      • Сохранение состояния                       │
        │                                                     │
        │      ⚠️ Выполняется ТОЛЬКО ОДИН РАЗ!              │
        │      ⚠️ Нет гарантии вызова при аварийном         │
        │         завершении (kill -9)                      │
        └───────────────────────────────────────────────────┘
                              ↓
                    ┌─────────────────┐
                    │   УНИЧТОЖЕНИЕ   │
                    │  (garbage       │
                    │   collection)   │
                    └─────────────────┘
