```cpp
inline int Sum(int a, int b)
{
	return a + b;
}
```

Компилятор вставит эту функцию в код программы, вместо того, чтобы в будущем идти и искать определение этой функции.

Но работает не всегда, т.к. если функция большая и сложная то компилятор просто не будет этого делать. 

А ещё компилятор сам может вставлять маленькие функции в код программы и без явного использования `inline`.
