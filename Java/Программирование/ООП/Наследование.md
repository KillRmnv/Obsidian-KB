Механизм, при котором один класс может быть создан на основе другого. Новый класс автоматически получает поля и методы базового класса и может дополнять или изменять их.

Наследование используется для переиспользования кода и построения иерархий классов, отражающих отношение «является». При этом чрезмерное использование наследования может привести к жёсткой связности.
