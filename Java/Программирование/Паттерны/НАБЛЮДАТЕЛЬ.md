[[Поведенческий паттерн|Поведенческий шаблон проектирования]], который определяет зависимость "один ко многим" между объектами. Он позволяет одному или нескольким объектам (наблюдателям) автоматически получать уведомления об изменениях состояния другого объекта.

[YouTube](https://youtu.be/yqhITl97HwU?si=Vi4TozDJ3lgMQu2U)
***
## Пример

```python
class PostSaver:
	_observers: list['Observer']
	
	def save_post(self):
		database.save()
		
		event = PostEvent("Данные о посте",...)
		for observer in self.__observers:
			observer.accept(event)

class Observer(ABC):
	@abstractmethod
	def accept(self, event: Event):
		pass
		
class NotificationObserver(Observer):
	def accept(self, event: Event):
		notification()
		
class RecommendationObserver(Observer):
	def accept(self, event: Event):
		recommend()
		
class AnalyticsObserver(Observer):
	def accept(self, event: Event):
		analyse()
```
