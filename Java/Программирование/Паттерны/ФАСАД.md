[[Структурный паттерн|Структурный паттерн проектирования]], который предоставляет простой интерфейс к сложной системе классов, библиотеке или фреймворку.
***
Предоставляет унифицированный интерфейс к группе интерфейсов подсистемы. Фасад определяет высокоуровневый интерфейс, упрощающий работу с подсистемой.
***
## Пример

```python
# Подсистемы (сложные компоненты)
class CPU:
    def start(self):
        print("CPU started")

    def execute(self, data):
        print(f"Executing {data}")

class Memory:
    def load(self, data):
        print(f"Loading {data} into memory")

class HardDrive:
    def read(self, file):
        print(f"Reading {file} from hard drive")

# Фасад — упрощённый интерфейс
class ComputerFacade:
    def __init__(self):
        self.cpu = CPU()
        self.memory = Memory()
        self.hard_drive = HardDrive()

    def start_computer(self, file):
        self.cpu.start()          # Запуск CPU
        self.memory.load(file)    # Загрузка в память
        self.hard_drive.read(file)  # Чтение с диска
        self.cpu.execute(file)    # Выполнение программы
```
