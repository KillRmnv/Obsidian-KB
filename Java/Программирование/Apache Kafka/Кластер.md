**Кластер** — набор брокеров, работающих вместе
- Один из брокеров кластера также выполняет роль **контроллера (controller)** — управляет распределением разделов, мониторит состояние брокеров.


все брокеры имеют одинаковую конфигурацию и топики.

При выходе из строя одного брокера все остальные брокеры хранят те же данные, что и вышедший из строя брокер -- [[репликация]]
