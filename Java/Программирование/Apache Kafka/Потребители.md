- Приложения, которые читают события из топиков.
    
- Читают данные **по своему усмотрению** — сами управляют своим оффсетом (в отличие от классических брокеров сообщений, где сообщение удаляется после отправки).
    
- Могут читать с любого места (с начала, с конца, с конкретного оффсета).
