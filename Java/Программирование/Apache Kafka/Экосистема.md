- **Kafka Connect:** Фреймворк для надежного и масштабируемого подключения Kafka к внешним системам (базы данных, облачные хранилища, HDFS и т.д.). Есть множество готовых коннекторов (JDBC, Elasticsearch, S3).
    
- **Kafka Streams:** Библиотека (не отдельный сервис) для построения приложений потоковой обработки на Java/Scala. Позволяет выполнять операции `filter`, `map`, `aggregate`, `join` над потоками данных из Kafka.
    
- **ksqlDB:** База данных для потоковой обработки на основе SQL. Позволяет писать запросы к данным в Kafka, как к таблицам в обычной БД.
    
- **REST Proxy:** Позволяет взаимодействовать с кластером Kafka через HTTP API, что удобно для клиентов не на JVM.
    
- **Schema Registry** (от Confluent, не входит в Apache): Централизованное хранилище схем данных (Avro, Protobuf, JSON Schema) для обеспечения совместимости форматов сообщений между производителями и потребителями.
