**Создатель**
***
Задаёт вопрос кто должен отвечать за создание нового экземпляра некоторого класса и назначает эту обязанность подходящему объекту.

Класс должен создавать экземпляры другого класса, если он:

1. Агрегирует или содержит его.
2. Активно использует его.
3. Обладает данными для его инициализации.
