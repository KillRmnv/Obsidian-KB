## Магические методы

### `__get__`

 *Используется для получения значения атрибута.*
### `__set__`

 *Используется для установки нового значения атрибута.*
### `__delete__`
 *Используется для удаления атрибута.*
###  `__set_name__`
 *Используется для захвата имени атрибута, как оно определено в классе владельца (классе, в котором определено свойство).*
