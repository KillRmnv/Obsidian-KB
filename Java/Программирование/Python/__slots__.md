Магический атрибут класса `__slots__` в [[Язык программирования Python|Python]] позволяет ограничить набор атрибутов, которые могут созданы в экземплярах данного класса. 

```python
class Person:
	__slots__ = ('name', 'age', 'email')
	
	def __init__(self, name, age, email):
		self.name = name
		self.age = age
		self.email = email

# Создание объекта
person = Person("Иван", 30, "ivan@example.com")
print(person.name)  # Иван
print(person.age)   # 30

# Попытка добавить новый атрибут вызовет ошибку
person.address = "Москва"  # AttributeError
```

### Преимущества
1. **Экономия памяти** - объекты не имеют `__dict__`
2. **Быстрый доступ к атрибутам**
3. **Защита от опечаток** в именах атрибутов
### Ограничения:

- Нельзя добавлять новые атрибуты динамически
- Наследование требует осторожности
- Несовместимость с некоторыми библиотеками

## Property
Несмотря на то, что **`slots`** ограничивает набор доступных атрибутов объекта до заранее определенного списка, их можно использовать вместе со свойствами

```python
class Rectangle:
    __slots__ = '__width', 'height'

    def __init__(self, a, b):
        self.width = a
        self.height = b

    @property
    def width(self):
        return self.__width

    @width.setter
    def width(self, value):
        print('setter called')
        self.__width = value
```
## Наследование
При простом наследовании вся функциональность `slots` для дочернего класса пропадает и вновь появляется возможность задавать любые атрибуты для класса.
```python
class Square(Rectangle):
    pass
```

*Теперь у экземпляра Square можно создавать новые имена атрибутов, которые отсутствуют в  `__slots__`  и также можно обращаться к атрибуту `__dict__`*
***
Чтобы в дочернем классе не пропадала функциональность `__slots__` родительского класса, нужно просто прописать свой __slots__
```python
class Rectangle:
    __slots__ = '__width', 'height'

class Square(Rectangle):
    __slots__ = tuple()
```
*Все атрибуты, которые вы укажите в `__slots__` дочернего класса, будут добавлены к именам из родительского  `__slots__`*
