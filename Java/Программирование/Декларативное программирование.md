Подход, при котором программа описывает **что нужно получить**, а не как именно это сделать.

Разработчик формулирует правила, ограничения или выражения, а способ их выполнения определяется средой выполнения или языком. Управление потоком и состоянием максимально скрыто.

```sql
SELECT SUM(value)
FROM numbers
WHERE value % 2 = 0;
```
