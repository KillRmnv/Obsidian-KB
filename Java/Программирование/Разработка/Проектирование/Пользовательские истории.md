## Критерии хорошей истории

- **I** \[Independent] - независима от других историй, то есть истории могут быть реализованы в любом порядке.
- **N** \[Negotiable] - обсуждаемая, отражает суть, а не детали. Не содержит конкретных шагов реализации.
- **V** \[Valuable] - ценная для клиентов, бизнеса и стейкхолдеров.
- **E** \[Estimable] - оцениваемая по сложности и трудозатратам.
- **S** \[Small] - компактная, может быть сделана командой за одну итерацию.
- **T** \[Testable] - тестируемая, имеет критерии приёмки.
