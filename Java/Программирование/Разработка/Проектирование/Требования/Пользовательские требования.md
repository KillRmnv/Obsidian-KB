Пользовательские требования описывают **что система должна предоставлять конечным пользователям**.

- Формулируются на языке пользователя, без программных деталей.
- Фокус на удобстве и функциональности для пользователей.
- Являются основой для проектирования интерфейсов и взаимодействия.

*«Пользователь должен иметь возможность создавать отчёты за выбранный период»*
