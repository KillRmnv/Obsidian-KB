Элемент объектной модели, позволяющий объектам разных классов **реагировать на одинаковые сообщения (вызовы методов) по-разному**. То есть один интерфейс может иметь несколько реализаций.

- **Обеспечивает гибкость и расширяемость системы.**
- **Позволяет писать обобщённый код, не зависящий от конкретного класса объекта.**
- **Упрощает замену и добавление новых типов без изменения существующего кода.**

Полиморфизм позволяет объектам **выполнять одно и то же действие по-разному**, сохраняя единый интерфейс.
