Крайне понравился брутализм в дизайне бюро, цветовой фильтр-создает нужную атмосферу.  Есть несколько действительно крутых локаций(пепельница).  Карта ужасна.  Головоломки не трудные, а заковыристые.  По геймплею-динамичный боевик с способностями.  В арсенале Джесси присутствует деформирующеесе оружие, которое может принимать форму ракетницы, пистолета, пистолета-пулемёта, обреза, дальнобойный мощный ствол, гранатамет(в длс).  Практически эффективными мне показались только 3 гана.  Оружие можно апгрейдить, открывая новые ячейки улучшений(не влияют на геймплей).  Но самым интересным элементом боевики являются способности.  Они также открываются по ходу сюжета.  Дэш, телекинез, щит, рукопашка(телекинезом), захват врагов, парение в воздухе.  Привносят динамики в геймплей, правда рукопашка бесполезна(только если броню сносить ею).  Значительным минусом является форсирование игрока играть агрессивно при малом количестве здоровья.
