В **Zen Browser** (как и в Firefox) есть встроенная функция ручного добавления поисковиков. Вот полная инструкция и готовые данные для всех движков из вашего списка.

### 📋 Инструкция: как добавить вручную
1.  Откройте **Настройки** (значок шестеренки или `Ctrl+,`).
2.  Перейдите в раздел **Поиск** → **Поисковые ярлыки** (Search Shortcuts).
3.  Нажмите кнопку **Добавить** (Add).
4.  Заполните поля, используя данные из таблицы ниже.
5.  Нажмите **Сохранить**.

> **Важно:** В поле URL обязательно должен быть символ `%s` — на его место браузер подставит ваш поисковый запрос.

---

### 🔧 Готовые данные для копирования

| Поисковик | Имя (Name) | Ключевое слово (Keyword) | URL с `%s` (вставьте это в поле URL) |
|-----------|------------|--------------------------|--------------------------------------|
| **Perplexity** | `Perplexity` | `p` | `https://www.perplexity.ai/search?q=%s` |
| **Bing** | `Bing` | `b` | `https://www.bing.com/search?q=%s` |
| **SearXNG** | `SearXNG` | `sx` | `https://searx.be/search?q=%s` |
| **Dogpile** | `Dogpile` | `d` | `https://www.dogpile.com/serp?q=%s` |
| **WolframAlpha** | `WolframAlpha` | `w` | `https://www.wolframalpha.com/input?i=%s` |
| **GitHub** | `GitHub` | `gh` | `https://github.com/search?q=%s&type=repositories` |
| **Codeberg** | `Codeberg` | `cb` | `https://codeberg.org/search?q=%s` |
| **Archive.org** | `Archive.org` | `arc` | `https://archive.org/search.php?query=%s` |
| **arXiv** | `arXiv` | `arx` | `https://arxiv.org/search/?query=%s&searchtype=all` |
| **Google Scholar** | `Google Scholar` | `gs` | `https://scholar.google.com/scholar?q=%s` |
